Sobre el proyecto

Este proyecto fue desarrollado por estudiantes de la UMSA de la carrera de Informática, con el propósito de visualizar el nivel de un tanque de gasolina mediante una interfaz moderna, clara y adaptable a distintos dispositivos.

El sistema permite mostrar los litros disponibles y para cuantos autos alcanza el combustible en tiempo real, facilitando el monitoreo del estado del tanque de una manera automática desde cualquier punto del mundo donde se encuentre el usuario (siempre y cuando tenga acceso a internet).

El proyecto fue realizado con el objetivo de proporcionar una solución eficiente y accesible para el monitoreo de niveles de gasolina para ver cual gasolinera tiene combustible disponible.
